# Email-Recon
### Version
0.0.1 (Beta)
### Installation
Clone repository
```sh
$ git clone https://github.com/matiasmenares/Email-Recon
```
## Getting started
Email Recon 0.0.1 (Beta) works with Python 2.7 You can run it with:
```sh
$ python email-recon.py -h
```
Command:
  - -g Generate shell
  - -p Setting Password
  - -u Url to connect
